# hello_world

Toolchain smoke test for the Spotpear C3 Mini. Prints chip info (cores, features,
silicon revision, flash size) and free heap, then restarts after 10 s.

From ESP-IDF `examples/get-started/hello_world`, CI harness files removed.

## Build and flash

```sh
. ~/esp/esp-idf/export.sh
idf.py set-target esp32c3
idf.py build
idf.py -p /dev/ttyACM0 flash monitor
```

Exit monitor with `Ctrl-]`.

`sdkconfig` is generated by `set-target` and not committed — defaults are fine for
this app.
